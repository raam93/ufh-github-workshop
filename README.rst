Building a Simple Professional Portfolio on GitHub 
===================================================
Resources for the **Build Free Professional Portfolios on GitHub** workshop at University Family Housing, University of Toronto.
 
Files
-----
 
- ``index.html`` — step-by-step workshop guide
- ``template.html`` — live demo of the finished portfolio
- ``explainer.html`` — plain-English walkthrough of every line of code
 
Usage
-----
 
Host this repo on GitHub Pages at ``yourusername.github.io/ufh-github-workshop``.
